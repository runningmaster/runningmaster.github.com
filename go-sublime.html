Sublime Text 2 как редактор для Go

Установка плагина GoSublime в текстовый редактор Sublime Text 2 позволяет наделить последний некоторыми полезными IDE-подобными функциями, как-то:

  • автозавершение кода (это делает Gocode)
  • переход к определению функции в исходном коде пакета (это делает MarGo)
  • показ документации к функции
  • список объявлений
  • отображение мест ошибок
  • автоформатирование кода при сохранении
  • выполнение кода
  • и прочая, прочая
$ go get -u github.com/nsf/gocode
$ go get -u github.com/DisposaBoy/MarGo
$ cd ~/.config/sublime-text-2/Packages
$ git clone git://github.com/DisposaBoy/GoSublime

Эх, еще бы добавить в редактор virtual space (обещают в 3 версии), без которого мне совсем плохо (привычка, выработанная тремя пятилетками в редакторе Delphi)…
