Элементарный генератор блога на Go

В качестве упражнения по изучению языка программирования Go сделал себе возможность генерации страниц для этого сайта.

Мои требования к процессу написания текстов для публикации в блоге удовлетворены чуть более, чем полностью:

  1. Sublime Text 2 - написание текста в правильном редакторе.
  2. Markdown - простое и быстрое оформление текста.
  3. Git - хранение текста в правильной системе контроля версий.
  4. Go - моментальное превращение текста в посты для блога.

Результатом доволен. Теперь все стало значительно проще, чем когда тоже самое делал при помощи Pelican. На досуге портирую старые тексты сюда.
