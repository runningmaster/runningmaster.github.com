Альтернатива Django|Rails на Go

Намедни открыл для себя один занимательный проект - высокоуровневый веб-фреймворк Go-start, который позиционируется как аналог Django для Python или Rails для Ruby, но только написан на Go и имеет в некоторой степени отличную от первых философию.

Фреймворк имеет спонсора и очень активно развивается, разгоняя ветер перемен базой данных MongoDB, которая используется по умолчанию.
